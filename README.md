# Customer Shopping Behaviour Analysis

## Project Overview
This project analyzes customer shopping behavior using SQL, Python, and Power BI. The objective is to identify customer purchasing patterns and generate business insights through data analysis and visualization.

## Files Included
- customer_shopping_behavior.csv - Dataset
- customer_behaviour_sql_queries.sql - SQL queries for analysis
- Customer_Shopping_Behaviour_Analysis.ipynb - Python analysis notebook
- customer_behaviour_dashboard.pbix - Power BI dashboard

## Tools Used
- SQL
- Python (Pandas, Matplotlib)
- Power BI

## Key Insights
- Customer purchase trends
- Spending patterns
- Popular product categories
- Customer segmentation analysis

## Dashboard Features
- Interactive visualizations
- Sales and customer insights
- Category-wise analysis
- Trend analysis
